What this repository contains



Scripts to reproduce every number, table, and figure in the paper, from raw data

retrieval through to the final figures. The pipeline has two data streams:





Stream A — training corpus. General FAERS reports pulled by receive date

(2025 Q1–Q4), leakage-audited, split temporally, used to train the clinical

seriousness classifier.

Stream B — held-out novel-drug panel. Six first-in-class drugs, excluded

entirely from training, used to test generalisation and to compare the

classical model against an LLM.

Data availability



All source data are public and are retrieved by 01\_pull\_quarters.py and

02\_extract\_panel.py from the openFDA drug-event endpoint

(https://api.fda.gov/drug/event.json). Derived data files are not committed;

they are regenerated by running the pipeline.



Reproducibility



A fixed random seed (42) is used for model fitting, bootstrap resampling, and all

stochastic components. Reported figures are reproducible from these scripts with

the library versions listed above.
